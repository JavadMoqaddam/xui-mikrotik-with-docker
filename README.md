#### Usage

1. **Install Docker:**

   ```sh
   bash <(curl -sSL https://get.docker.com)
   ```

2. **Clone the Project Repository:**

   ```sh
   git clone https://github.com/idaniali/xui-mikrotik-with-docker.git
   cd xui-mikrotik-with-docker
   ```

3. **Start the Service:**

   ```sh
   docker compose up -d
   ```
